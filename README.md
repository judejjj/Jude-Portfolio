## 🌟 Overview

A lightweight, fully responsive portfolio site rendered with Flask templates and hand-written CSS — no frontend framework, no bloat. Built to showcase projects, research, and background in a way that feels intentional rather than auto-generated.

## 💎 Features

| | |
|---|---|
| 🎨 **Premium UI/UX** | Deep dark mode with neon accents (cyan, purple, pink) and frosted-glass panels |
| 🌀 **Dynamic Animations** | Floating 3D avatar, animated gradients, smooth hover transitions |
| 📩 **Serverless Contact Form** | Submits silently to Google Forms — no redirect, custom success modal instead |
| 📱 **Fully Responsive** | Seamless across desktop, tablet, and mobile |
| ⚙️ **Flask Backend** | Lightweight, fully customizable Python backend with clean template rendering |

## 🛠️ Tech Stack

- **Backend** — Python, Flask
- **Frontend** — HTML5, Vanilla CSS3 (custom variables, Flexbox, Grid)
- **Icons** — FontAwesome 6
- **Typography** — Google Fonts (*Outfit*, *Inter*)

## 📂 Project Structure

```
Personal-Portfolio-Website/
├── app.py                 # Main Flask application and routing
├── requirements.txt       # Python dependencies
├── static/
│   ├── css/style.css      # Custom styling, dark mode, glassmorphism
│   ├── images/            # 3D avatar and visual assets
│   └── js/                # Client-side interactivity
└── templates/
    ├── base.html          # Master layout template
    ├── index.html         # Hero section & landing page
    ├── about.html         # Bio, technical arsenal, certifications
    ├── projects.html       # Portfolio projects & GitHub links
    └── contact.html       # Contact form (Google Forms integration)
```

## 🎨 Customization

- **Colors & Theming** — all primary colors, glass opacities, and gradients live in the `:root` variables at the top of `static/css/style.css`
- **Hero Image** — swap `static/images/3d_image_me.png` for your own avatar; the floating animation applies automatically
- **Contact Form** — update the Google Form action URL and `entry.*` field names in `templates/contact.html`
